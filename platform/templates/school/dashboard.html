  • إضافة صف — استخدم النموذج في الأعلى لإنشاء صف دراسي جديد. اختر المستوى التعليمي المناسب، ثم حدد المعلم المسؤول عن الصف والمعلم الاحتياطي إن وجد، ثم اضغط "إضافة صف".
  • طلاب — اضغط على هذا الزر داخل بطاقة الصف لعرض قائمة بجميع الطلاب المسجلين في هذا الصف. يمكنك من هناك إضافة طلاب جدد أو إزالتهم.
  • تقويم — يفتح تقويم الصف الذي يعرض الأحداث والمواعيد المهمة. يمكنك إضافة أحداث جديدة مثل الاختبارات أو الأنشطة وتعديلها أو حذفها.
  • مراقبة — يفتح صفحة المراقبة المستمرة حيث يمكنك إدخال درجات الطلاب لكل مادة ومتابعة أدائهم خلال الفترات الدراسية المختلفة.
  • واجب — يتيح لك إنشاء واجبات منزلية جديدة للصف وتحديد تاريخ التسليم، ومتابعة حالة تسليم الطلاب لكل واجب.
  • دعم — يعرض المواد الداعمة المرتبطة بمستوى هذا الصف، مثل الملفات التعليمية والموارد الإضافية التي تم رفعها مسبقاً.
  • إعلان — يسمح لك بإرسال إعلان خاص بهذا الصف فقط. سيظهر الإعلان للطلاب والمعلمين المرتبطين بالصف عند دخولهم إلى لوحة التحكم.
  • حضور — يفتح صفحة تسجيل الحضور والغياب اليومي للطلاب. حدد التاريخ ثم ضع علامة بجانب كل طالب حاضر واترك الغائبين بدون علامة.
  • تعديل / حذف — زر "تعديل" يسمح لك بتغيير اسم الصف أو المستوى أو المعلم المسؤول. زر "حذف" يزيل الصف نهائياً مع جميع بياناته — يرجى الحذر عند استخدامه.

تفاصيل الصف:
المنهج الخاص بالمستوى
{% if role == 'local_admin' %}
  • إضافة مستوى — اكتب اسم المستوى التعليمي الجديد في الحقل (مثلاً: "المستوى الأول" أو "المستوى المتقدم") ثم اضغط "إضافة مستوى". المستويات تُستخدم لتصنيف الصفوف وربطها بالمناهج المناسبة.
  • مواد داعمة — اضغط على هذا الزر داخل بطاقة المستوى لرفع ملفات تعليمية داعمة (مثل ملفات PDF أو صور أو مقاطع صوتية). هذه المواد ستكون متاحة لجميع الصفوف المرتبطة بهذا المستوى.
  • مناهج — يفتح قسم إدارة المناهج الدراسية لهذا المستوى. يمكنك إنشاء مجموعات مناهج وإضافة مواد دراسية داخل كل مجموعة، وتعديل أو حذف المواد والمجموعات حسب الحاجة.
  • تعديل / حذف — زر "تعديل" يسمح بتغيير اسم المستوى. زر "حذف" يزيل المستوى نهائياً — تأكد أولاً من عدم وجود صفوف مرتبطة به قبل الحذف.
{% endif %}
{% if role == 'local_admin' %}
  • إضافة معلم — استخدم النموذج في الأعلى لتسجيل معلم جديد. أدخل البريد الإلكتروني (سيكون اسم المستخدم لتسجيل الدخول)، واسم المعلم الكامل، وكلمة مرور مؤقتة. بعد الإضافة، يمكن للمعلم تسجيل الدخول والوصول إلى صفوفه.
  • معلومات — اضغط على هذا الزر بجانب اسم المعلم لعرض تفاصيله الكاملة، بما في ذلك رقم الهاتف والملاحظات الخاصة والتنبيهات المرسلة إليه. يمكنك من هنا أيضاً تعديل هذه البيانات.
  • تعديل / حذف — زر "تعديل" يسمح بتغيير اسم المعلم أو بريده الإلكتروني أو كلمة المرور. زر "حذف" يزيل حساب المعلم نهائياً — سيتم إلغاء ربطه بجميع الصفوف المرتبطة به، لذا يُنصح بتعيين معلم بديل أولاً.
{% endif %}
{% if role == 'local_admin' %}
  • إضافة طالب — استخدم النموذج في الأعلى لتسجيل طالب جديد. أدخل اسم المستخدم (يستخدمه الطالب لتسجيل الدخول)، واسم الطالب الكامل، وكلمة مرور. بعد الإضافة، يمكنك إلحاق الطالب بصف من تبويب "الصفوف".
  • بحث — اكتب في حقل البحث للعثور على طالب معين بسرعة. يمكنك البحث بالاسم الكامل أو الجزئي أو باسم المستخدم. النتائج تظهر فوراً أثناء الكتابة.
  • تجميع حسب الصف — فعّل خانة "تجميع حسب الصف" بجانب حقل البحث لعرض الطلاب مصنفين حسب صفوفهم بدلاً من القائمة العادية. هذا مفيد لمعرفة توزيع الطلاب على الصفوف.
  • القدرات — اضغط على هذا الزر بجانب اسم الطالب لعرض صفحة تفصيلية بقدراته وتقييماته في المواد المختلفة، بما في ذلك الشارات والإنجازات التي حصل عليها.
  • بطاقة الدرجات — يعرض بطاقة درجات شاملة للطالب تتضمن جميع درجاته في المواد والاختبارات والمراقبة المستمرة، مع المعدل العام والملاحظات.
  • تعديل / حذف — زر "تعديل" يسمح بتغيير اسم الطالب أو اسم المستخدم أو كلمة المرور. زر "حذف" يزيل الطالب نهائياً من النظام مع جميع درجاته وبياناته — يرجى الحذر الشديد عند استخدامه.
{% endif %}
{% if role == 'local_admin' %}
  • إعلان عام — اضغط هذا الزر لإنشاء إعلان عام جديد أو تعديل الإعلان الحالي. الإعلان يظهر لجميع المستخدمين (معلمين وطلاب) عند دخولهم لوحة التحكم. يمكنك تحديد تاريخ انتهاء صلاحية الإعلان.
  • شارات مميزة — ينقلك إلى صفحة إدارة الشارات المميزة. يمكنك إنشاء شارات جديدة بأيقونات وألوان مختلفة، وتعيينها للطلاب المتميزين كنوع من التحفيز والمكافأة.
  • إدارة المدراء المحليين — ينقلك إلى صفحة إدارة حسابات المدراء المحليين. يمكنك إضافة مدراء جدد أو تعديل صلاحياتهم أو إزالتهم. المدراء المحليون لديهم نفس صلاحياتك في إدارة المدرسة.
  • معلومات المدرسة — ينقلك إلى صفحة تعديل بيانات المدرسة الأساسية، مثل اسم المدرسة وشعارها (اللوغو) ومعلومات الاتصال. هذه البيانات تظهر في واجهة جميع المستخدمين.
  • تحميل نسخة احتياطية — يقوم بتنزيل ملف مضغوط يحتوي على نسخة كاملة من قاعدة البيانات وجميع الملفات المرفوعة (صور، مستندات، إلخ). يُنصح بأخذ نسخة احتياطية بشكل دوري للحفاظ على بياناتك.
  • إرسال النسخة بالبريد — يرسل النسخة الاحتياطية مباشرة إلى بريدك الإلكتروني المسجل في النظام. مفيد إذا كنت تريد الاحتفاظ بنسخة خارجية دون تنزيل الملف يدوياً.
